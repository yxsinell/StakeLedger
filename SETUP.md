# StakeLedger - Frontend Setup

## Version

- Version: v1.0.0 (primera version del frontend)

## Requisitos

- Node.js 18+
- Bun (package manager)

## Instalacion

```bash
bun install
```

## Desarrollo

```bash
bun run dev
```

## Build

```bash
bun run build
```

## Estructura

- src/app: App Router
- src/components/ui: Design system (shadcn/ui)
- src/components/layout: Sidebar y header
- src/lib: utils + tipos

## Notas

- Tailwind CSS v3 configurado
- Paleta basada en el logo (public/imageSL.png)
